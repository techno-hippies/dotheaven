# Web App

SolidJS + Vite web client.

## Location
- Source: `apps/web/src/`

## Common Commands (from repo root)
- `bun dev` — run web dev server.
- `bun build` — production build.
- `bun check` — type checks across repo packages/workspaces.

## Runtime Configuration
- Uses `apps/web/.env` for frontend env vars.
- See root `README.md` and service docs for endpoint dependencies.
