# Apps

User-facing clients in this repository.

## Directories
- `web/` — SolidJS web app.
- `desktop/` — Rust + GPUI native desktop app (Linux/macOS/Windows).
- `android/` — Kotlin + Jetpack Compose Android app.

## Notes
- iOS Swift client is planned but not present yet.
